# Public Interface

All traffic into the platform or hosted sites runs through a Kubernetes ingress and nginx proxy.

# Platform

The platform itself is made up of a simple UI www.wbstack.com (opens new window) and API www.wbstack.com/api (opens new window).

# Sites

All Sites exist under a single domain (or subdomain).

For example for the site addshore-alpha.wiki.opencura.com (opens new window):

  • MediaWiki
    • addshore-alpha.wiki.opencura.com/w (opens new window)
    • addshore-alpha.wiki.opencura.com/wiki (opens new window)
  • Query
    • addshore-alpha.wiki.opencura.com/query (opens new window)
    • addshore-alpha.wiki.opencura.com/query/sparql (opens new window)
  • Miscellaneous Tools
    • addshore-alpha.wiki.opencura.com/tools/cradle (opens new window)
    • addshore-alpha.wiki.opencura.com/tools/quickstatements (opens new window)
    • addshore-alpha.wiki.opencura.com/tools/widar (opens new window)
